# Old / Unused Files

This folder contains files and directories that are not used when running the application on localhost (e.g. when using `php artisan serve` and `npm run dev` / `npm run build`).

- **dist/** — Alternate build output (the app uses `public/build/` from Vite).
- **scripts/** — Empty / unused scripts.
- **old/** — Legacy unused directory.
- **docker-compose.yml** — Docker setup; not used when running locally without Docker.

These can be kept for reference or removed.
