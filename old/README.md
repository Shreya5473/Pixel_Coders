# Old / unused files

This folder is for **deprecated or unused** project files.

- Move here any code or assets you no longer use (e.g. old seeders, legacy views, unused config).
- Do **not** move files that are still referenced by the app (routes, config, packages) or the build (Vite, themes).
- See **`PROJECT_STRUCTURE.md`** at repo root for the full Frontend / Backend / Old layout.

If this folder is empty, it means no legacy files have been archived yet.
